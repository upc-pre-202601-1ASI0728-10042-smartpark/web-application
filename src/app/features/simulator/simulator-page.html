Simulador IoT

Emite alertas de humo y simula movimiento de vehículos para demostrar el sistema en vivo (SignalR, dashboard y gemelo 3D).

¿Cómo usar el simulador?

Este panel reemplaza a los sensores IoT físicos para demostrar el sistema en vivo durante las entrevistas de validación. Sigue estos pasos:

  1. 1
    Alerta de humo. Elige una zona y el nivel de humo en ppm, y pulsa “Disparar alerta”. O activa “Humo automático” para emitir una alerta aleatoria cada 7 s.
  2. 2
    Ocupación. Pulsa “Simular movimiento” para que entren y salgan vehículos (cambia plazas libres/ocupadas). O activa “Ocupación automática” cada 4 s.
  3. 3
    Dónde verlo. Las alertas de humo aparecen como toast y en la campana 🔔, y se listan en “Alertas de humo”. La ocupación se refleja en Dashboard y Zonas, y en el Gemelo 3D (pulsa allí “Actualizar ocupación” para ver las plazas cambiar de color).
